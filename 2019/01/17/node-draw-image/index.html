    2. gm - node
      gm 是通过 node 调用 GraphicsMagick 和 ImageMagick 来实现绘图功能的。其中，GraphicsMagick “短小精悍”，号称是“图像处理领域的瑞士军刀”, 是从 ImageMagick 5.2.2 中独立出来的。
      gm 支持链式调用，基本功能类似 canvas api，但更强大。可查看官方文档深入研究。

      使用方法：
